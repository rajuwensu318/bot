What is Cryptocurrency?

Types of Cryptocurrency

Bitcoin (BTC)

The first cryptocurrency and digital gold used for storing value.

Ethereum (ETH)

A platform that allows apps and smart contracts to run on blockchain.

Stablecoins

Coins like USDT or USDC that are tied to real currencies like USD.

Altcoins

All other cryptocurrencies beyond Bitcoin, like SOL, XRP.
